3D Transform in CSS ?

CSS 3D Transforms allow you to create three-dimensional effects on web elements by manipulating their position and perspective in a 3D space. This is achieved using properties such as transform, perspective, and transform-style.

Key Properties

  • transform: This property is used to apply 3D transformations like rotateX(), rotateY(), rotateZ(), translateZ(), and scaleZ().
  • perspective: This property defines the perspective from which the 3D elements are viewed, giving a sense of depth.
  • transform-style: This property specifies how nested elements are rendered in 3D space, with the value preserve-3d allowing child elements to maintain their 3D position.

Example

.box {
  width: 100px;   
  height: 100px;
  background-color: lightblue;
  transform: rotateX(45deg) rotateY(30deg) translateZ(50px);
  perspective: 500px;
  transform-style: preserve-3d;
}

In this example, a box is rotated along the X and Y axes and translated along the Z-axis, creating a 3D effect.
