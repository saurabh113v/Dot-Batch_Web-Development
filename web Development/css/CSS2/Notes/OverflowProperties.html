CSS Overflow Properties

CSS overflow properties are used to control how content that exceeds the dimensions of its container is handled. These properties help manage the visibility and behavior of overflowing content, ensuring a better user experience on web pages.

Common CSS Overflow Properties

  • overflow: This property specifies what happens to content that overflows the bounds of its container. It can take values such as visible, hidden, scroll, and auto.
    • Example values:
    • visible: The overflow is not clipped and will be visible outside the element's box.
    • hidden: The overflow is clipped, and the rest of the content will be invisible.
    • scroll: The overflow is clipped, but a scrollbar is added to see the rest of the content.
    • auto: If the content overflows, a scrollbar will be added; otherwise, it behaves like visible.
  • overflow-x: This property controls the horizontal overflow of content. It can be set to the same values as the overflow property but only affects the horizontal axis.
  • overflow-y: This property manages the vertical overflow of content. Similar to overflow-x, it can be set to control only the vertical axis.
