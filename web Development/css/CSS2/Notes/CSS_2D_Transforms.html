CSS 2D Transforms

CSS 2D transforms allow you to manipulate the position, size, and shape of HTML elements in a two-dimensional space. These transformations can be applied using the transform property in CSS.

Common 2D Transform Functions

  • Translate: Moves an element from its current position by a specified distance along the X and Y axes. Example: transform: translate(50px, 100px);
  • Rotate: Rotates an element around a fixed point (default is the center of the element) by a specified angle. Example: transform: rotate(45deg);
  • Scale: Resizes an element by a specified factor along the X and Y axes. Example: transform: scale(1.5, 2);
  • Skew: Distorts an element along the X and Y axes by a specified angle. Example: transform: skew(20deg, 10deg);
