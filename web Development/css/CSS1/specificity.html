What is Specificity?

Specificity is a set of rules that browsers use to determine which CSS rule to apply when multiple rules could apply to the same element. It is calculated based on the types of selectors used in the CSS.

Example of Specificity

Consider the following CSS rules:

  p {
    color: blue;
  } 
  .highlight {
    color: red;
  }
  #unique {
    color: green;
  }

In this example, if a paragraph has the class "highlight" and the id "unique", the text color will be green because the ID selector has higher specificity than the class selector and the element selector.

Specificity Hierarchy

  • Inline styles (e.g., style="color: blue;") - highest specificity
  • ID selectors (e.g., #unique)
  • Class selectors (e.g., .highlight), attribute selectors (e.g., [type="text"]), and pseudo-classes (e.g., :hover)
  • Element selectors (e.g., p, h1) and pseudo-elements (e.g., ::before)
  • Universal selector (*) - lowest specificity

Draw a Hierarchy of specificity

Inline styles > ID selectors > Class/Attribute/Pseudo-class selectors > Element/Pseudo-element selectors > Universal selector

Understanding specificity is crucial for writing effective CSS and ensuring that the desired styles are applied to HTML elements.
